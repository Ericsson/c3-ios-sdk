C3FileRef

@objc final public class C3FileRef: C3EventEmitter

Represents a reference to a file, which resides either locally, or at some remote peer.

If the file is available locally it can be accessed directly, otherwise it has to be downloaded first.

Version

1.0.0

Events

  • end: Emitted when the file download ends, both on success and failure. Receives C3FileRefEndReason instance.
  • error: Emitted when the file download ends before it is completed. Receives Error instance.
  • load: Emitted when the file download is completed successfully. Receives C3FileRef instance.
  • progress: Emitted each time the progress of the file download changes. Receives Float instance.
  • transfer: Emitted each time a remote peer starts downloading the file. Receives C3FileTransfer instance.
  • transfers: Emitted when the list of file transfers is changes. Receives [C3FileTransfer] instance.
