Client

final public class Client: EventEmitter

Represents a client.

Version

1.0.0

Events

  • event: Emitted when an event is received in any room. Receives Event instance.
  • event:<type>: Same as event, but listens to a specific event type.
  • call: Emitted when there is an incoming call to any room. Incoming calls are stopped and need to be started with call’s start() before a connection is set up. Receives Call instance.
  • invite: Emitted when a new room invite is received. The room will contain additional information, such as invitedBy. Receives Room instance.
  • rooms: Emitted whenever the list of rooms changes.
  • state: Emitted when the conneciton state of the client is changed.
