EventEmitter

@objc open class EventEmitter: NSObject

Base for classes which generate events to which one can bind to.

Version

1.0.0
  • off(_:target:callback:)

    Removes a listener associated with an event. At most one listener is removed, if the same listener has been added multiple times, only the first listener is removed.

    Version

    1.0.0

    Declaration

    Swift

    final public func off(
        _ eventType: String,
        target: AnyObject? = nil,
        callback: Selector? = nil)

    Parameters

    eventType

    The event type from which to remove the listener(s).
    target

    The target that should be removed. If omitted, all listeners for the event type are removed.
    callback

    The callback that should be removed. If omitted, all listeners for the target are removed.
  • on(_:target:callback:)

    Adds a new listener for an event. All listeners will be called in the order in which they were added. Adding an already added listener is not an error and will result in the listener being called multiple times.

    Version

    1.0.0

    Declaration

    Swift

    final public func on(
        _ eventType: String,
        target: AnyObject,
        callback: Selector)

    Parameters

    eventType

    The event to listen for.
    target

    An owner of the selector that will be called whenever the event is emitted.
    callback

    A selector that will be called whenever the event is emitted.
  • once(_:target:callback:)

    Works like on, but the listener is at most called once, and then removed.
