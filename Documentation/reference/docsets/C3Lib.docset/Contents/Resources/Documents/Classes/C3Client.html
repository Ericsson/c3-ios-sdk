C3Client

final public class C3Client: C3EventEmitter

Represents a client.

Version

1.0.0

Events

  • event:<type>: Same as event, but listens to a specific event type.
  • call: Emitted when there is an incoming call to any room. Incoming calls are stopped and need to be started with call’s start() before a connection is set up. Receives C3Call instance.
  • event: Emitted when an event is received in any room. Receives C3Event instance.
  • invite: Emitted when a new room invite is received. The room will contain additional information, such as invitedBy. Receives C3Room instance.
  • rooms: Emitted whenever the list of rooms changes.
  • state: Emitted when the conneciton state of the client is changed.

  • setAvatar(_:success:failure:)

    Set the avatar of the authenticated user.

    Typical usage is to first upload an image file using C3Client.uploadMedia(_:ofType:success:failure:), and then setting it as the avatar.

    There is no guarantee that client.user.avatar will be up to date if and when the returned promise is resolved. UI updates to reflect the new avatar should instead be triggered by the avatar event of the user.
