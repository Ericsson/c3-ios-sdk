C3DeviceSource

final public class C3DeviceSource: C3MediaNode

A media source that streams from user media devices.

Version

1.0.0

Events

  • error: Emitted if a stream could not be acquired. Receives Error instance. The node will only ever emit one of the stream and error events, and only once.
  • stream: Emitted when the stream has been acquired. Receives RTCMediaStream instance. The node will only ever emit one of the stream and error events, and only once.

  • mute

    Toggles muting of the media.

    When assigning to the property, both audio and video will be updated.

    Version

    v.0.0

    Example

    let source = C3DeviceSource()
    


    // both at once
source.mute = true
    


    // directly
source.muteVideo = true
